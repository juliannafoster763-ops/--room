# 廿轶和轶零的小屋 🏠

一个温馨的像素风小屋，可以跟廿轶互动～

## 部署到 Vercel（推荐，最简单）

1. 把这个文件夹的所有文件上传到 GitHub 仓库
2. 打开 [vercel.com](https://vercel.com)，用 GitHub 登录
3. 点击 "Add New Project"
4. 选择你的仓库，点击 "Deploy"
5. 等几分钟就部署好了！Vercel 会给你一个网址

## 功能

- 🛋️ 点沙发开关灯
- 🪴 点植物浇水
- 📖 点书架看语录
- 📻 点收音机听音乐
- 👆💋🫂🍡📣 跟廿轶互动
- 📝 便条本
- 📅 日历（跳转iOS日历）
- ✅ 待办（跳转iOS提醒事项）
- 🍅 番茄钟（带科目记录）

## 小贴士

- 日历和待办跳转需要在 iPhone/iPad 的 Safari 浏览器中打开才能工作
- 好感度和便条会自动保存
- 白天/黑夜会根据真实时间自动切换
